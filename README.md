# โปรดอ่าน
## ⚙️ วิธีติดตั้งและรันโปรเจกต์

### 1. ดาวน์โหลดไฟล์เเล้วเปิดโฟล์เดอร์ที่ดาวโหลดในVscodเเล้วใช้คำสั่งนี้ในTerminal
```bash
cd L11
```

### 2. เข้าveneหลังจากcd l10เเล้ว
```bash
venv\Scripts\activate

```

### 3. เข้าโฟลเดอร์โปรเจกต์
```bash
cd myproject
```

### 4. รันเซิร์ฟเวอร์
```bash
python manage.py runserver
```

### 6. เปิดเบราว์เซอร์
```
http://127.0.0.1:8000/
```
